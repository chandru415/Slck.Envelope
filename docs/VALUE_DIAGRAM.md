## ?? Developer Journey

### Step 1: One-Time Setup (5 minutes)

```csharp
// Program.cs (1 line!)
builder.Services.AddSlckEnvelopeObservability(builder.Configuration);
```

### Step 2: Configure (appsettings.json)

```json
{
  "SlckEnvelope": {
    "Observability": {
      "Enabled": true,
      "ActivitySourceName": "MyAPI"
    }
  }
}
```

### Step 3: Write Handlers (Forever After)

```csharp
// Every handler from now on:
public class MyQuery(
    ILogger<MyQuery> logger,
    ActivitySource activitySource,
    IRepo repo,
    SlckEnvelopeObservabilityOptions? options = null)
    : ObservableQuery<Data>(logger, activitySource, options)
{
    public override async Task<IResult> HandleAsync()
    {
        // JUST business logic - ZERO boilerplate!
        return Envelope.Ok(await repo.GetAsync());
    }
}
```

### Step 4: Enjoy Benefits

? **OTEL traces** - automatic  
? **Serilog logs** - automatic  
? **TraceId correlation** - automatic  
? **Error tracking** - automatic  
? **Consistent naming** - automatic  
? **Configuration** - appsettings.json
